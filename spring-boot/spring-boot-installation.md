# Installation in mac os

Using [homebrew](https://brew.sh/)

## Check java version :pencil:

```shell
$ java -version
```

## Install spring boot :rocket:

```shell
$ brew update
$ brew tap pivotal/tap
$ brew install springboot
```

## Check spring version :pencil:

```shell
$ spring --version
# Spring CLI v2.2.4.RELEASE
```
