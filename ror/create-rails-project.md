# Hello Rails project :rocket:

Create new app.

```shell
$ rails new appName [options]
```

If you want to build project with the other rails version.

```shell
$ rails _version_ new appName
```

Check directory tree.

```shell
$ tree -L 1
.
├── Gemfile
├── Gemfile.lock
├── README.md
├── Rakefile
├── app
├── babel.config.js
├── bin
├── config
├── config.ru
├── db
├── lib
├── log
├── node_modules
├── package.json
├── postcss.config.js
├── public
├── storage
├── test
├── tmp
├── vendor
└── yarn.lock
```

## Rails server

Move to root directory, start rails server.

```shell
$ rails server
```

Check local server, `http://localhost:3000/`
